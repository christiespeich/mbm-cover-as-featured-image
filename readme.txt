Use a book's cover image as the featured image of the page.

== Description ==

Use a book's cover image as the featured image of the page.  This would allow you to use it in sliders, for example.

Be aware that your theme may use the featured image in the templates and affect your book page.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/mbm-cover-as-featured-image` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
